HTML常用标签

Written by fujiaoning with ♥ on August 29, 2019 in

a标签用法

  • 属性
    1. href超链接
    2. target指定在哪里打开页面
    3. download理论上下载网页,不过很多情况下不支持
    4. rel=noopener防止bug

  • a的href取值
    1. 网址
      https://google.com
      http://google.com
      //google.com无协议网址
    2. 路径
      /a/b/c以及a/b/c
      index.html 以及./index.html
    3. 伪协议
      javascript:代码;
      mailto:邮箱
      tel:手机号
    4. id
      href=#xxx
  • 作用
    1. 跳转外部页面
    2. 跳转内部锚点
    3. 跳转到邮箱或电话等
  • a的target取值
    1. 内置名字
      _blank在空白页面打开
      _top在顶级区域打开
      _parent在上一级区域打开
      _self在当前区域打开
      以当前窗口打开为例
    2. 程序员命名
      window的name
      iframe的name

table标签用法

  • 相关标签

    1. thead头部
    2. tbody内容
    3. tfoot尾部
    4. tr其中的一行
    5. td数据

    6. th表头
      table里只能有三个标签

      <body>
<table>
<thead>
    <th></th>
    <th>包子</th>
    <th>烧卖</th>
    <th>小饼</th>
</thead>
<tbody>
    <tr>
        <th>肉馅</th>
        <th>7个</th>
        <th>2个</th>
        <th>5个</th>
    </tr>
    <tr>
        <th>素馅</th>
        <th>1个</th>
        <th>3个</th>
        <th>4个</th>
    </tr>
    <tr>
        <th>糖馅</th>
        <th>3个</th>
        <th>5个</th>
        <th>2个</th>
    </tr>
</tbody>
<tfoot>
    <tr>
        <th>总量</th>
        <td>11个</td>
        <td>10个</td>
        <td>11个</td>
    </tr>
</tfoot>
</table>
</body>

  • 样式

    1. table-layout宽度
    2. border-collapse合并边距
    3. border-spacing调整空隙

img标签

  • 作用
    发出get请求,展示一张图片
  • 属性
    alt/图片加载失败时显示描述
    height/高度,若只写高度,宽度会自适应
    width/宽度,若只写宽度,高度会自适应
    src图片地址
  • 事件
    onload/onerror监听图片是否加载成功
  • 响应式
    max-width:100%

form标签

  • 作用
    发get或post请求,然后刷新页面
  • 属性
    action请求后到哪个页面
    autocomplete自动填充
    method控制用get还是post请求
    target提交到哪个页面,哪个页面要刷新
  • 事件
    input里面不可以有其他标签,button可以
    onsubmit必须有type=“submit”

input标签

  • 作用
    让用户输入内容

  • 属性
    类型type

    1. button
    2. checkbox
    3. email
    4. file
    5. hidden
    6. number
    7. password
    8. radio
    9. search
    10. submit
    11. tel
    12. text

  • 其他
    name/autofocus/checked/disabled/maxlength/pattern/value/placehoder

  • 事件
    onchange/onfocus/onblur

  • 验证器
    html新增功能
