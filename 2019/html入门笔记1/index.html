WWW:World Wide Web

万维网WWW=URL+HTTP+HTML

HTML之父

Tim Berners-Lee,1990年左右基于互联网发明了万维网,用自己写的浏览器访问了自己的服务器,让每个人输入网址就能看到网页,2004年获得大英帝国爵级司令勋章.2017年被颁发图灵奖.

HTML起手

<!DOCTYPE html> # 文档类型
<html lang="en"> # 页面语言 中文通常改成zh-CN

<head>
    <meta charset="UTF-8"> # 支持所有语言标准
    <meta name="viewport" content="width=device-width, initial-scale=1.0"> # 防止页面缩放
    <meta http-equiv="X-UA-Compatible" content="ie=edge"> # 更新至最新页面版本
    <title>Document</title> # 网页标题
</head>

<body>

</body>

</html>
