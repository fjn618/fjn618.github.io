JS数据类型

Written by fujiaoning with ♥ on August 29, 2019 in

7种数据类型

  1. 数字number
    NaN(Not a Number)用来表示 无法表示的数字 NaN不等于NaN 64位浮点数
    符号占1位,指数占11位(-1023~1024),有效数字占52位(开头的1省略)
    最大值Number.MAX_VALUE:1.7976931348623157e+308
    最小值Number.MIX_VALUE:5e-324
    15位有效数字可以精确表示,或者16位有效数字,小于90开头
  2. 字符串string
  3. 布尔bool true和false
    1、只要“&&”前面是false，无论“&&”后面是true还是false，结果都将返“&&”前面的值;
    2、只要“&&”前面是true，无论“&&”后面是true还是false，结果都将返“&&”后面的值;
  4. 符号symbol
    引号不包含在字符串内.
    如果想储存’it’s ok’ 则写为
    ‘it\’s ok’
    “it’s ok”
    或改成反引号 it's ok
    转义
    \n 表示换行, \r表示回车, \t表示tab制表符, \ \表示 (斜杠\ )
    \uFFFF表示对应的Unicode字符 \xFF表示前256个Unicode字符
  5. 空undefined
  6. 空null
  7. 对象object
    总结:四基两空一对象
