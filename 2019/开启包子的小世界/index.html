如何用 hugo 搭建个人博客

  • 第一步

    1. 安装hugo
      https://github.com/gohugoio/hugo/releases

    2. 解压

    3. 把Hugo所在的文件夹路径添加到path

    4. 重启终端,输入 hugo version 查看版本号

  • 第二步

    1. 进入hugo官网 https://gohugo.io/

    2. 点击quick start

    3. 从step2开始复制代码到终端

    4. 输入 hugo server 预览网页
