如何用 hugo 搭建个人博客

Written by fujiaoning with ♥ on August 29, 2019 in

Hugo:Go语言实现的博客生成器

  • 第一步:下载hugo

    1. 安装hugo
      Mac安装方式
      brew install hugo
      hugo version
      Windows安装方式
      下载hugo_版本号_系统.zip 下载地址
      以我的版本为例

    2. 解压后得到hugo.exe文件

    3. 把hugo.exe所在的文件夹路径添加到path
      右键此电脑,进入属性,点击高级系统设置,选择环境变量,双击path,点击新建,将hugo.exe的路径粘贴后点击确定

    4. 重启终端,输入 hugo version 查看版本号

  • 第二步:搭建框架

    1. 创建一个目录并进入它

    2. 进入hugo官网后点击quick start

    3. 从step2开始复制代码到终端
      step2:hugo new site quickstart
      创建一个名为quickstart的仓库,建议将名字更改为你的用户名.github.io-creator
      step3:
      cd quickstart
      git init
      git submodule add https://github.com/budparr/gohugo-theme-ananke.git themes/ananke
      echo 'theme = "ananke"' >> config.toml
      本地仓库初始化,并使用ananke主题进行美化 更多主题
      step4:hugo new posts/my-first-post.md
      添加了名为my-first-post的一篇内容
      step5:hugo server -D启动hugo
      step6:打开config.toml编辑以下内容

      baseURL = "https://你的域名/"
languageCode = "zh-ch"
title = "你的网站名"
theme = "ananke"

      step7:hugo构建静态页面
